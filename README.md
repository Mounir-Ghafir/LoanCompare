# LoanCompare

A simple React app that helps you compare loan offers from different providers.

## Features

- View loan offers from different providers
- Click on any offer to see monthly payment and total cost
- Simple and easy to understand code

## How It Works

1. Shows loan offers with provider name, amount, and interest rate
2. Click an offer to see the payment details

## Tech Stack

- React
- Vite
- Plain CSS

## Getting Started

```bash
npm install
npm run dev
```
